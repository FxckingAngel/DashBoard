Moderation

Eli has simple yet powerful moderation commands to keep your server in check while not giving your staff too much power.

With the proper setup you can see all executed moderation commands and force your staff to moderate through Eli instead of default Discord perms. This allows you to easily spot abuse.

Eli has moderation commands ranging from kicking and giving warnings to mass deleting spam and temporarily muting/banning members.

Automation

Ever got tired of the basic ugly looking welcome message discord sends whenever some joins the server?

Or did you ever wish you could customize the message Discord sends when someone boosts the server?

Eli solves those problems with amazing looking welcome & goodbye messages as well as custom boost messages. You can also set up automatic reminders for disboard. You can find the commands to do these things in the automation section

Reaction Roles

Eli has some of the easiest and prettiest reaction roles on all of Discord. Use the easy set up from eli rr create or turn any message into a reaction role menu with eli rr addrole [message ID] [Emote] [Role].

Find out more about how you can make and edit existing reaction role menus at the Commands List

Leveling

Eli has a very powerful yet free leveling system! Users gain experience for talking and level up after gaining enough experience. You can configure every aspect of it like how much xp they gain per minute, which roles Eli should give for reaching a certain level and how those roles stack up.

Check your levels with eli rank and eli profile or see where you rank in the server or global leaderboards with the
eli leaderboard (global) command

Gifs & Roleplay

With over 50 commands, going from hugging, cuddling & patting to biting & shooting others, Eli has the widest variety of roleplay commands of all Discord bots!

Eli tracks how many times you've performed an action or had it happen to you so you can see exactly how many times you've hugged others or got headpats.

Unlike other big bots all gifs in Eli are handpicked and she uses an algorithm to send them in a randomized order while never sending the same gif twice! This all to ensure the best experience for your members and keep it safe for all ages.
